# Сопутствующие материалы (Supplementary Materials)

В этой директории находятся дополнительные ресурсы и материалы для курса "Разработка приложений с использованием LLM".

## 📁 Структура директории

Здесь будут размещены:

- **Шаблоны документов** — готовые шаблоны для технической документации, промптов, чеклистов
- **Дополнительные примеры кода** — расширенные примеры, не вошедшие в основные модули
- **Полезные скрипты** — утилиты для автоматизации разработки и тестирования
- **Справочные материалы** — шпаргалки, таблицы сравнения моделей, гайды по настройке
- **Презентации и схемы** — визуальные материалы для лучшего понимания архитектурных решений

## 📝 Как использовать

Материалы в этой директории предназначены для дополнения основных модулей курса. Вы можете:

1. Использовать шаблоны для ускорения разработки своих проектов
2. Изучать дополнительные примеры для углубления понимания тем
3. Применять скрипты для автоматизации рутинных задач
4. Обращаться к справочным материалам при возникновении вопросов

## 🔄 Обновления

Директория будет регулярно пополняться новыми материалами по мере развития курса.
