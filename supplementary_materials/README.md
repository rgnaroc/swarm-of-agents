# Сопутствующие материалы (Supplementary Materials)

В этой директории находятся дополнительные ресурсы и материалы для курса "Разработка приложений с использованием LLM".

## 📁 Структура директории

| Файл | Описание |
|------|----------|
| `github_guide.md` | Подробное руководство по работе с GitHub: настройка репозиториев, ветвление, pull request'ы и совместная разработка |
| `python_colab_guide.md` | Руководство по использованию Python и Google Colab для разработки и тестирования LLM-приложений |

## 📝 Как использовать

Материалы в этой директории предназначены для дополнения основных модулей курса:

1. **github_guide.md** — используйте при работе с репозиториями, настройке окружения и совместной разработке
2. **python_colab_guide.md** — обращайтесь при запуске кода в облачной среде или локальной разработке на Python
