# Swarm of Agents

Репозиторий посвящён изучению и разработке мультиагентных систем на основе искусственного интеллекта.

## 📚 Содержание курса

Этот проект представляет собой учебный курс по созданию агентов ИИ и мультиагентных систем. Материалы организованы по модулям:

| Модуль | Файл | Описание |
|--------|------|----------|
| 0-1 | [module_0_1_beginner_guide.md](module_0_1_beginner_guide.md) | Введение для начинающих |
| 2-3 | [module_2_3_tools_memory.md](module_2_3_tools_memory.md) | Инструменты и память агентов |
| 4-5 | [module_4_5_frameworks_multiagent.md](module_4_5_frameworks_multiagent.md) | Фреймворки и мультиагентные системы |
| 6-7 | [module_6_7_integration_security.md](module_6_7_integration_security.md) | Интеграция и безопасность |
| 8 | [module_8_production_deployment.md](module_8_production_deployment.md) | Production и развёртывание |
| 9 (бонус) | [module_9_bonus_advanced.md](module_9_bonus_advanced.md) | Продвинутые темы |

## 📋 Дополнительные материалы

- **[ai_agents_course_plan.md](ai_agents_course_plan.md)** — Полный план курса по агентам ИИ
- **[final_roadmap_checklists.md](final_roadmap_checklists.md)** — Итоговая дорожная карта и чек-листы

## 🚀 Быстрый старт

1. Начните с модуля 0-1 для ознакомления с основами
2. Последовательно проходите модули от простого к сложному
3. Используйте чек-листы из `final_roadmap_checklists.md` для отслеживания прогресса

## 🎯 Целевая аудитория

- Разработчики, интересующиеся ИИ-агентами
- Исследователи в области мультиагентных систем
- Все, кто хочет создать собственного ИИ-агента
