## ✅ ЧЕК-ЛИСТЫ ПО УРОВНЯМ

### Уровень 1: Новичок (Недели 1-3)

```
□ Установлен Python 3.10+
□ Создано виртуальное окружение
□ Получен API-ключ OpenAI
□ Сделан первый API-запрос
□ Написано 10 разных промптов
□ Создан простой чат-бот
□ Освоены техники: Zero-shot, Few-shot, CoT
□ Понимание: temperature, max_tokens, top_p
□ Создан персональный ассистент с контекстом
```

**Мини-проект**: Персональный ассистент-консультант

### Уровень 2: Начинающий (Недели 4-5)

```
□ Освоен Function Calling
□ Создано 3+ инструмента для агента
□ Реализован цикл: мышление → действие → наблюдение
□ Подключён веб-поиск
□ Работа с файлами (чтение/запись)
□ Понимание embeddings
□ Установлена ChromaDB
□ Реализован RAG
□ Агент отвечает по загруженным документам
```

**Мини-проект**: Агент-консультант по PDF-документации

### Уровень 3: Средний (Недели 6-8)

```
□ Созданы цепочки в LangChain
□ Использован LCEL синтаксис
□ Создан агент с инструментами в LangChain
□ Построен граф в LangGraph
□ Реализованы условные переходы
□ Запущен CrewAI с несколькими агентами
□ Создана команда AutoGen с диалогом
□ Реализован Supervisor pattern
□ Понимание: ReAct, Human-in-the-loop
```

**Мини-проект**: Команда аналитиков (исследователь → аналитик → писатель)

### Уровень 4: Продвинутый (Недели 9-10)

```
□ Интеграция с браузером (Playwright)
□ Подключение к базе данных
□ Реализован Text-to-SQL
□ Создан Telegram-бот
□ Работа с email
□ Защита от prompt injection
□ Sandboxing для кода
□ Rate limiting
□ FastAPI для агента
□ Streamlit интерфейс
□ Docker-контейнеризация
□ Деплой на облако
```

**Мини-проект**: Личный секретарь (почта + календарь + Telegram)

### Уровень 5: Эксперт (Бонус)

```
□ Fine-tuning модели (LoRA)
□ Локальные модели (Ollama)
□ Мультимодальность (изображения)
□ MCP-протокол
□ Self-reflection агент
□ Оптимизация стоимости
□ Fallback-модели
□ Мониторинг и алерты
```

**Финальный проект**: Production-ready SaaS с AI-агентом
